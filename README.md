## 📖 About the Project
Eaye Ease is a hackathon project aimed at enhancing **eye care accessibility and patient experience**.  
Our solution provides a **user-friendly interface** that simplifies navigation, improves usability, and ensures a seamless digital journey for patients and hospital staff.
